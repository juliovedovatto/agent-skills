# generate-image

Generates images from text prompts via the Replicate API.

## Required environment variable

```bash
export REPLICATE_API_TOKEN=your_token_here
```

This skill will not run without `REPLICATE_API_TOKEN`. Get a token from your [Replicate account](https://replicate.com/account/api-tokens).

## Setup

```bash
npm install
```
